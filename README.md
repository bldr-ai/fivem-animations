1. Download the `.ycd` file
2. Place it in your resource's `stream` folder
3. Reference it in your script using the clip dictionary and clip name
```lua
TaskPlayAnim(ped, "cwalk", "anim", 1.0, 1.0, -1, 1, 0, false, false, false)
```
